# NEXUS — The AI Command Center

Unify OpenCode, ChatGPT+, Gemini Pro, and Go into a single devastatingly effective development workflow.

## Stack

| Engine | Role |
|--------|------|
| OpenCode | AI coding agent — the conductor |
| Go | Validated, benchmarked models |
| ChatGPT+ | Frontier LLMs (GPT-4o, o3) |
| Gemini Pro | 1M context, multimodal |

## Features

- **Agent Skills** — load-on-demand SKILL.md power-ups
- **Multi-Agent System** — Build, Plan, Explore, Scout
- **MCP Servers** — connect to databases, APIs, SaaS
- **Custom Tools** — define your own functions
- **75+ Providers** — any model, any provider, zero lock-in

## Workflow

1. **Plan** with Claude (via Go)
2. **Build** with GPT-4o
3. **Analyze** with Gemini Pro
4. **Ship** with OpenCode
